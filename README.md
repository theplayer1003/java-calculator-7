# java-calculator-precourse
___

## InputHandler
> 입력을 처리하는 클래스
### Reader
> 입력을 수행하는 클래스


## OuputHandler
> 출력을 처리하는 클래스
### Printer
> 출력을 하는 클래스


## Calculator
> 계산을 처리하는 클래스
### TargetNumber
> 숫자를 관리하는 클래스


## DelimiterParser
> 입력 받은 문자열을 파싱하는 클래스
### Delimiter
> 구분자를 관리하는 클래스


## CalculateService
> 도메인 흐름을 담당하는 클래스
 

## CalculateController
> 입출력과 함께 도메인 흐름을 담당하는 클래스
